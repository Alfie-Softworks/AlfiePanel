AlfiePanel – Custom Commands API

This document is a deep, practical guide to writing custom commands.

1. Execution Pipeline

Player → ChatHandler → Prefix → Parser → Lookup → Rank Check → Run() → Response

2. Command Structure

{
    Name = "example",
    Aliases = {"ex"},
    Rank = 1,
    Run = function(ctx)
    end
}

3. Command Fields

Name: Unique identifier

Aliases:

Aliases = {"tp", "bring"}

Rank: Required power

4. Context (ctx)

FieldMeaning
SpeakerPlayer running command
ArgsArguments
SettingsLive settings
UtilsHelpers
ReplyFeedback

5. Reply

ctx:Reply("Success")
ctx:Reply("Error", "error")

6. Argument Safety

local target, err = ctx.Utils.findPlayerByNameOrId(ctx.Args[1])
if not target then
    ctx:Reply(err, "error")
    return
end

7. Rank Safety

local ok, err = ctx.Utils.canChangeRank(myPower, targetPower)
if not ok then
    ctx:Reply(err, "error")
    return
end

8. DataStore Safety

Utils.dataStoreCallWithRetry(function()
end)

Common Mistakes

  • Using Players:GetPlayers()
  • No validation
  • Unsafe rank changes
